# 🏎️ Car Racing
A Formula 1 style racing game in Python using Pygame.
## 🌟 Features
- **F1 theme:** Use of liveries and aesthetics of famous cars.
- **Motion mechanics:** Realistic car control on the track.
- **Design patterns:** The project is built taking into account architectural standards for easy addition of new tracks or car types.
## 🛠️ Technologies
- **Language:** Python 3.x
- **Library:** Pygame.
